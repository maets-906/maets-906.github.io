hexo new post "新建博客文章名" 是一个命令，用于在您的 Hexo 博客项目中创建一篇新文章。这个命令会在 source/_posts 目录下生成一个新的 Markdown 文件，文件名就是您在命令中指定的文章名。您可以编辑这个 Markdown 文件来撰写您的博客文章内容。当您运行 hexo generate（或简写为 hexo g）命令时，Hexo 将会把这篇文章生成为一个静态 HTML 文件，然后可以通过部署到托管服务来展示给访问者。 ——来自GPT-4
